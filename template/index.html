Como jogar

Não são aceitas palavras no plural.


O objetivo do jogo é adivinhar uma palavra dentro de um limite de tentativas. Quanto menos tentativas o jogador utilizar, mais pontos ele ganhará. Durante o jogo, o jogador terá a opção de solicitar dicas, desde que tenha pontos suficientes para adquiri-las, já que as dicas são compradas por meio de pontos ganhos durante a rodada.


O perfil do jogador, incluindo os pontos acumulados e o usuário criado, será disponibilizado apenas durante a sessão de jogo. Ao sair do jogo, o jogador perderá todos os pontos e terá que criar um novo perfil.

Ao confirmar uma palavra as cores irão indicar se a letra existe ou não na palavra em questão, dependendo da cor o jogador ganhará uma quantia de pontos por cada letra existente na palavra.

  • - Esta cor indica uma letra existente na palavra que está na posição correta (vale 2 pontos).
  • - Esta cor indica uma letra existente na palavra, porém está na posição incorreta (vale 1 ponto).
  • - Esta cor indica uma letra inexistente na palavra (0 pontos).

Durante a rodada o jogador poderá "comprar" dicas utilizando os pontos da rodada. Metade dos pontos serão descontados por cada dica que for solicitada pelo jogador.

  • book

  • keyboard_backspace
  • border_color

Para liberar as dicas o jogador deve ter uma quantia mínima de pontos, sendo necessario 2 pontos para pedir uma letra e 4 pontos para pedir uma dica de texto.


Desistência

  • sentiment_very_dissatisfied

Caso o jogador não saiba a palavra, ele terá a opção de desistir da rodada atual e iniciar uma nova. Entretanto, os pontos conquistados na rodada não serão contabilizados em sua pontuação total/rank. Além disso, o jogador perderá 10% dos pontos totais/rank. Se o jogador não acertar a palavra, ocorrerá uma redução de 15% em seus pontos totais/rank.


Rank

Os pontos só serão contabilizados no rank caso o jogador acerte a palavra.
